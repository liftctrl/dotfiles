# dotfiles

## Install Base Packages

- alacritty 
- rofi
- ranger
- picom
- feh
- nerd-fonts
- starship
- eza

```bash
sudo pacman -S alacritty rofi ranger picom feh nerd-fonts starship eza
```

yay -S catppuccin-gtk-theme-macchiato

```bash
sudo cp -r dotfiles/{alacritty,i3,redshift,gtk-3.0,picom,rofi} .config/
```
